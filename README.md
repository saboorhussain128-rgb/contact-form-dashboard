# 🚀 Contact Form Project (Node.js + Express + MongoDB)

A full-stack contact form application with admin authentication and message management system.

---

## 📌 Features

### 🧾 Contact Form
- First name, last name, email, phone, message
- Server-side validation
- Character counter
- Success & error messages
- Data stored in MongoDB

### 🔐 Admin System
- Secure login system
- Session-based authentication
- Logout functionality
- Protected admin routes

### 📊 Admin Dashboard
- View all submitted messages
- Search functionality
- Pagination support
- Delete messages
- Card-based modern UI

### 🎨 UI/UX
- Dark modern theme
- Responsive design
- Clean card layout
- Animated buttons and hover effects

---

## 🛠 Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- EJS Templates
- CSS3
- Express Session
